## 寻路速查

| 要做什么 | 从哪里开始 |
|----------|------------|
| 第一次使用 | `README.md` → `CLAUDE.md` / `AGENTS.md` |
| 填写个人背景 | `01-关于我/人物档案.md` |
| 查看卡片示例 | `02-知识卡片/index.md` |
| 投递原始素材 | `素材箱/_待处理/` |
| 了解维护流程 | `AI协作工具/skills/` |
| 启用 Hook | `.claude/settings.example.json` + `AI协作工具/hooks/CLAUDE.md` |
| 查看方法论 | `原理/工作台方法论.md` |

## 安全边界

- 真实 API Key、topic ID、账号信息只保存在本机环境变量或未提交的 `.env`。
- 仓库不会默认启用 Hook；复制示例配置后才会执行。
- 分享或推送前，必须扫描 Git 当前快照与完整历史中的敏感信息。
