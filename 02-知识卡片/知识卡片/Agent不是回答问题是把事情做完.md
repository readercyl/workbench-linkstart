# Agent不是回答问题是把事情做完

钱塘江鲤的核心定义：Agent = 帮我们把事情做完的AI员工。"做完"和"回答"有本质差异。

Chatbot（豆包/ChatGPT）：被动响应，问一句答一句，给建议不给结果。Agent：目标驱动，端到端交付。让它找便宜机票→它搜索、比价、看日期、比较航司、查天气、推荐酒店、甚至直接下单。

Agent的工作循环：思考→行动→观察→再思考→再行动→再观察（React机制），直到目标达成。Claude Code之所以厉害，核心就是这个循环。
