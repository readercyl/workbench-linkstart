# 四层工具选择逻辑

按场景特征从稳定到灵活，四层选择不可倒过来：

| 层级 | 场景特征 | 推荐方案 | 稳定性 |
|:--:|:--|:--|:--:|
| 1 | 流程完全固定，输入-处理-输出明确，无 AI 决策需求 | 脚本 | 99%+ |
| 2 | 流程固定但有 AI 判断需求（摘要、分类、生成） | Claude Agent SDK + 脚本 | 高 |
| 3 | 需要人机频繁交互、串联多种通讯工具、定时任务 | OpenClaw | 中 |
| 4 | 快速验证想法、给客户做 Demo、启发客户理解 AI 能力 | Skill | 低 |

> 钱塘江鲤："这个顺序不能倒过来。很多人先选了 OpenClaw 再给它找用法，方案一定会变重。"
