# 05-项目运营·子宪法

> 版本：v1.1 · 遵从根宪法 v2.0 · 2026-06-15

## 目录结构

```
05-项目运营/
├── 01-AI自媒体/              ← AI 自媒体账号运营策略和内容
├── 02-虚拟产品/              ← 虚拟产品策划、定价、销售
├── 03-公众号流量主矩阵/       ← 公众号矩阵运营
├── 项目总览.md               ← 所有项目的全局视图
└── CLAUDE.md
```

## 使用约定

- AI 讨论项目方向时，先读「项目总览.md」了解全局
- 涉及具体项目操作时，进入对应子目录
- 账号信息属于敏感数据，AI 不主动暴露给外部

## 铁律

- 不在此目录存放知识/认知/方法论（归 02-知识卡片 或 03-认知工坊）
- 不在此目录存放内容素材（归 04-内容生产）
- 新增项目子目录后更新本文件的目录结构节
