# 06-对外分享·子宪法

> 版本：v1.2 · 遵从根宪法 v2.0 · 2026-07-14

分享版当前只保留本子宪法。新增对外文档或展示类 Skill 后，再按实际文件更新目录结构。

## 目录定位

对外分享放的是**面向外部读者**的文档和工具。判断标准：产出物是给「用户以外的人」看的（分享/演示/教程/小说）→放这里。

## 维护约定

- 内部文档改了 → 检查对外版本是否需要同步更新
- 对外版本保持独立，不引用内部路径（`02-知识卡片/` 等）
- 文件名带日期的是特定版本快照，不带日期的是持续更新的最新版
- 展示类 Skill 如需加入，放在本目录的 `skills/` 下

## 铁律

- 不在此目录存放内部协作文档（内部协作归 AI协作工具 或 00-运转日志）
- 新增对外文档后更新本文件目录结构节
